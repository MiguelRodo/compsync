# Homebrew Installation

Install `compsync` via Homebrew (macOS / Linux):

```bash
brew tap MiguelRodo/compsync https://github.com/MiguelRodo/compsync
brew install compsync
```

Or install directly from the formula file:

```bash
brew install --formula homebrew/compsync.rb
```

## Requirements

- Homebrew
- Python 3 (installed automatically as a dependency)
- Git

## Usage

```bash
compsync update
compsync --help
```
